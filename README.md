# Practise_Swift

Swift learning practice repository.

## Topics Covered

- **Basics** — Constants, variables, type inference, type annotations, string interpolation
- **Operators** — Assignment, arithmetic, remainder, unary, compound, comparison, ternary, nil-coalescing, range, and logical operators
- **Strings and Characters** — String literals, multiline strings, Unicode, string interpolation, indices, substrings, and comparison
- **Collection Types** — Arrays, sets, set operations (union, intersection, subtracting, symmetricDifference), set membership, dictionaries
- **Optionals & Optional Chaining** — Optional types, nil-coalescing, optional chaining
- **Protocols** — Protocol declaration, conformance, protocol as type, protocol inheritance
- **Extensions** — Extending Int, String, and Double with computed properties and methods
- **Generics** — Generic functions, generic types (Stack), type constraints (Equatable), associated types
